# Ali Wael — Portfolio

Personal portfolio site: a cinematic, editorial single-page showcasing my public work — web platforms, mobile apps and backends.

**Stack**

- [Next.js 16](https://nextjs.org) (App Router) + TypeScript
- Tailwind CSS v4
- Framer Motion (reveals, preloader) + GSAP ScrollTrigger (image parallax)
- Lenis smooth scrolling
- Fraunces + Inter via `next/font`

**Develop**

```bash
npm install
npm run dev
```

**Deploy**

Push to GitHub and import into [Vercel](https://vercel.com) — no env vars needed.

Project screenshots live in `public/shots/` (captured from live deployments and local runs of each repo).
